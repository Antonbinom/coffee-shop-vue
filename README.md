# coffee-shop

## Описание

Pet-проект Сайт кофе

## Функционал

- Переход по страницам без перезагрузки
- Динамический рендеринг данных
- Загрузка товаров с сервера
- Поиск и фильтр товаров(подгружаемых с сервера)
- Плавный скролл

## Стек технологий

- Vue2
- Vue-router
- Vuex
- Vue-cli
- Axios
- Debounce
